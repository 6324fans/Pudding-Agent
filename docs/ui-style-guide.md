# Pudding-Agent UI 风格提示词文档

> 本文档定义了 Pudding-Agent 的 UI 设计规范，用于指导后续页面开发保持一致的视觉风格。

---

## 一、整体风格定位

**关键词：** 深色主题 / 现代极简 / 科技感 / 沉浸式 / 精致细腻

整体采用 **深色优先（dark-first）** 的设计理念，以深邃的近黑色为底色，搭配微妙的紫蓝色调作为品牌色，营造出专业、沉浸的 AI 编程工具氛围。所有交互元素都追求圆润、柔和、精致的质感。

## 三、字体系统

| 用途 | 字体栈 | 说明 |
|------|------|------|
| 无衬线（主体） | `"Inter", "SF Pro Display", "Helvetica Neue", sans-serif` | UI 文字、正文 |
| 等宽字体 | `"JetBrains Mono", "Geist Mono", "SF Mono", monospace` | 代码、路径、技术标识 |

**字号体系：**
- 标题：16–20px，font-weight: 600，tracking-tight
- 正文：13–14px，font-weight: 400，leading-relaxed
- 辅助：11–12px，font-weight: 400–500
- 微型标签：9–10px，大写，letter-spacing: 0.12–0.15em

## 四、圆角 & 间距

### 圆角规范
- 小元素（标签、徽章）：`4–6px`
- 按钮、输入框：`8–10px`
- 卡片、面板：`12px`（rounded-xl）
- Logo 图标：`16px`（rounded-2xl）

### 间距节奏
- 基础单位：4px
- 组件内边距：`12–24px`（px-3 到 px-6）
- 组件间间距：`8–24px`（gap-2 到 gap-6）
- 区块间间距：`24–32px`（gap-6 到 gap-8）

## 六、组件设计规范

### 6.1 侧边栏（Sidebar）

**品牌区域：**
- 顶部放置 Logo 图标 + 应用名 + 副标题
- Logo 为 32×32px 圆角方块，背景使用品牌渐变（`#818cf8 → #a78bfa`）
- 内部为白色代码图标（`</>`）
- 应用名 14px 加粗，副标题 10px 弱化色

**会话列表：**
- 项目名用 10px 大写、加宽字距的微型标签
- 每个会话为圆角行，带状态指示灯（6px 圆点）：
  - 空闲态：弱化灰点
  - 选中态：品牌色点 + 品牌柔色背景 + 辉光边框
  - 运行中：琥珀黄脉冲动画
  - 错误：珊瑚红点
  - 完成：翡翠绿点
- 悬停时背景变为 `surface-2`
- 新建会话按钮：虚线圆点 + 文字

**底部区域：**
- "新项目" 按钮：全宽、边框按钮，悬停时边框加深
- 版本号小字居中

### 6.2 顶部栏（Topbar）

- 左侧：项目名称（16px 加粗）+ 会话 ID 徽章（等宽字体，surface-2 背景）
- 右侧：主题切换器 + 新建会话按钮 + 设置图标
- 支持拖拽移动窗口（`-webkit-app-region: drag`）
- 主题切换器为分段控件，选中项使用品牌渐变背景

### 6.3 会话头部（SessionHeader）

- 左侧面包屑：项目名 / 会话ID（等宽字体）
- 中部工具栏：外部应用启动器 + 终端按钮（圆角边框组）
- 右侧状态指示器：
  - 模型名称徽章（surface-2 背景，11px）
  - 权限模式文字
  - 推理强度（绿色脉冲点 + 文字）
  - 规划模式（紫色脉冲点）
  - 流式状态（琥珀黄脉冲点）
  - 队列计数（品牌色半透明背景）
  - 上下文用量进度条（品牌渐变/红色）

### 6.4 聊天视图（ChatView）

**用户消息：**
- 左侧 3px 品牌色竖线 + 左内边距
- 14px 正文，行高宽松

**AI 回复：**
- 14px 正文，行高宽松（leading-relaxed）
- Markdown 渲染：
  - 标题：sans-serif 字体，加粗（非衬线体）
  - 行内代码：品牌色文字 + surface-2 背景 + 边框 + 圆角
  - 代码块：surface-2 背景 + 圆角 12px + 表面-3 头部栏（语言标签 + 复制按钮）
  - 表格：surface-3 表头 + 圆角容器
  - 引用块：3px 品牌色左边框 + 品牌辉光背景 + 右侧圆角
  - 链接：品牌色 + 下划线

**思考过程块：**
- 折叠式卡片，surface-2 背景
- 紫色状态点（运行中带脉冲动画）
- 展开后显示等宽文字的思考内容

**工具调用卡片（ToolCardShell）：**
- 圆角 12px 卡片
- 状态色边框（运行中=琥珀、完成=默认、错误=红色）
- 头部：状态点 + 展开箭头 + 工具名 + 参数详情（等宽字体）
- 运行中带微弱投影

### 6.5 输入框（Composer）

**主输入区域：**
- 圆角 12px 容器，surface-2 背景，默认边框
- **聚焦时**：边框变为品牌色 + 辉光阴影（核心视觉反馈）
- 左侧可选代码图谱状态指示器
- 文本区域透明背景，14px
- 底部状态栏（11–12px）：
  - 权限模式（彩色状态点 + 文字 + 下拉）
  - 推理强度（状态点 + 文字 + 下拉）
  - 规划模式切换
  - 分支切换器
  - IDE 连接状态
  - 右侧：模型选择器

**发送按钮：**
- 品牌渐变背景（`#818cf8 → #a78bfa`）
- 白色文字 + 发送图标
- 辉光阴影
- 禁用态降低透明度

**停止按钮：**
- 红色边框 + 红色文字
- 悬停时填充红色背景 + 白色文字

### 6.6 检查器（Inspector）

**折叠态（48px）：**
- 垂直排列图标按钮
- 悬停时背景变色
- 徽章：16px 圆形，品牌色背景，白色数字

**展开态：**
- 头部：标题 + 关闭按钮
- 图标导航栏：选中项带品牌柔色背景 + 辉光边框
- 内容区：各面板（会话、用量、任务、团队、队列、文件）
- 左侧拖拽手柄调整宽度

### 6.7 项目首页（ProjectPage / 空状态）

**有项目时：**
- 项目信息卡片（辉光边框）：项目名 + 路径 + 模型徽章 + 会话数
- 2 列网格：最近会话、MCP 健康、用量概览、任务概览
- 卡片悬停时边框加深

**无项目时（欢迎页）：**
- 居中布局
- 大号品牌渐变 Logo 图标（64×64px）
- 欢迎文字 + 描述
- 品牌渐变主按钮（"新建项目"）

### 6.8 错误卡片（ErrorCard）

- 3px 红色左边框 + surface-2 背景
- 红色脉冲状态点 + 大写错误类别标签
- 重试按钮（绿色）+ 关闭按钮
- 下方错误详情（等宽字体）

### 6.9 通知提示（Toast）

- 右上角弹出，滑入动画
- 圆角 8px，3px 彩色左边框
- 成功=绿色、错误=红色、信息=品牌色
- 毛玻璃效果（backdrop-blur）

### 6.10 终端面板（TerminalPanel）

- 深色背景（`#0a0b0f`），与页面背景一致
- 光标颜色使用品牌色（`#818cf8`）
- 完整的暗色终端配色方案
- 顶部拖拽调整高度手柄
- 头部：终端标签 + 关闭按钮

## 七、动效规范

| 动效名称 | 用途 | 参数 |
|----------|------|------|
| `pulse` | 状态指示灯（运行中、流式生成） | Tailwind `animate-pulse` |
| `glow-pulse` | 品牌辉光呼吸 | 2s ease-in-out infinite，box-shadow 变化 |
| `toast-slide-in` | 通知从右侧滑入 | 180ms ease-out，translateX(20px→0) |
| `bubble-pop-in` | 元素弹出 | 160ms ease-out，scale(0.96→1) |
| `shimmer` | 加载骨架屏 | 2s infinite，渐变位移 |
| 聚焦过渡 | 输入框边框+阴影 | transition-colors + transition-shadow |
| 悬停过渡 | 按钮/卡片背景 | transition-all duration-150 |

## 八、交互状态规范

### 按钮状态
| 状态 | 样式 |
|------|------|
| 默认 | 边框按钮或渐变填充 |
| 悬停 | 背景加深 / 透明度变化 |

| 聚焦 | 品牌色边框 + 辉光阴影 |
| 错误 | 红色边框 |
| 禁用 | 降低透明度 |

### 列表项状态
| 状态 | 样式 |
|------|------|
| 默认 | 透明背景 |
| 悬停 | surface-2 背景 |
| 选中 | 品牌柔色背景 + 辉光边框 + 品牌色文字 |
| 活跃（运行中） | 脉冲状态点 |

## 十、设计原则

1. **深色优先** — 所有设计从暗色主题出发，亮色主题为辅助
2. **微妙层次** — 通过 `surface` 的三级递进建立空间层次感，而非依赖强对比
3. **品牌色克制** — 品牌紫蓝色仅用于关键交互节点（按钮、聚焦、选中），避免大面积使用
4. **状态驱动** — 所有元素通过状态点（绿/黄/红/紫）直观传达系统运行状态
5. **圆润柔和** — 所有交互元素使用 8–12px 圆角，营造友好、现代的感觉
6. **动效服务功能** — 脉冲动画表示"正在运行"，过渡动画表示"状态变化"，不为装饰而动效
7. **信息密度适中** — 12–14px 正文、宽松行高、合理留白，兼顾信息量和可读性

## 十一、Prompt 模板（用于 AI 生成页面时引用）

当你需要 AI 生成符合此设计系统的页面时，可以使用以下提示词模板：

```
请按照以下 UI 设计规范生成页面代码：

**设计风格：** 深色主题现代极简科技风

**色彩：**
- 背景：#0a0b0f（页面）、#12131a（卡片/侧边栏）、#181a24（输入框/代码块）、#1e2030（悬停态）
- 文字：#e4e5eb（主）、#a0a3b1（次）、#6b6e7e（弱化）
- 品牌色：#818cf8，渐变 #818cf8→#a78bfa（135°）
- 功能色：#34d399（成功）、#fbbf24（警告）、#f87171（错误）、#a78bfa（规划）
- 边框：rgba(255,255,255,0.06) 默认、rgba(255,255,255,0.12) 强调、rgba(99,102,241,0.3) 辉光

**字体：**
- 主体：Inter / SF Pro Display / sans-serif
- 代码：JetBrains Mono / Geist Mono / monospace
- 正文 13-14px，辅助 11-12px，微型标签 9-10px 大写宽字距

**圆角：** 按钮/输入框 8-10px，卡片 12px，小元素 4-6px

**阴影：** 0 1px 3px rgba(0,0,0,0.4) 基础、0 8px 32px rgba(0,0,0,0.5) 弹出、0 0 20px rgba(99,102,241,0.15) 辉光

**交互：**
- 输入框聚焦时边框变品牌色 + 辉光阴影
- 按钮使用品牌渐变背景 + 白色文字
- 状态指示灯用 6-7px 圆点 + 脉冲动画
- 所有过渡使用 transition-all 150-200ms
- 选中项使用品牌色半透明背景（rgba(99,102,241,0.15)）

**布局：** 左侧 260px 侧边栏 + 中间弹性内容区 + 右侧 48px 折叠/320px 展开的检查器

使用 Tailwind CSS 4 和 CSS 变量（--bg, --surface, --text, --accent 等）。
```
